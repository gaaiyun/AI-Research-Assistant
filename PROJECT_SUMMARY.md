## 🚀 使用方法

### 1. 安装依赖

```bash
cd ai-research-assistant
pip install -r requirements.txt
```

### 2. 运行测试

```bash
python run_tests.py
# 或
pytest tests/ -v
```

### 3. 启动应用

```bash
streamlit run dashboard.py
```

应用将在 http://localhost:8501 启动

## 🏗️ 技术架构

### 核心技术栈

- **前端**: Streamlit 1.28+
- **数据处理**: Pandas, NumPy
- **可视化**: Plotly 5.17+
- **PDF 解析**: PyPDF2, pdfplumber
